# Prompt-Writer-Prototipe

Um simples projeto para escritores, basicamente te da alguns temas, generos e um tom, e te pede para escrever alguma historia com base naquilo.

feito para ajudar a destravar a escrita e treinar quando não se estiver trabalhando em algo
